Enable browser notifications

If the test notification in the extension settings didn't appear on screen, your operating system is likely blocking Chrome's notification permission. Follow the steps for your OS below.

🍎 macOS

  1. Open System Settings (Apple menu → System Settings).
  2. Click Notifications in the sidebar.
  3. Scroll down and click Google Chrome.
  4. Set the alert style to Banners or Alerts (not Off).
  5. Make sure Allow Notifications is toggled on.
  6. Go back to the extension settings and click Send test notification again.
If you see two "Google Chrome" entries, enable notifications for both — one is the main browser and the other handles background processes like extensions.
Also check that Focus (Do Not Disturb) is not active — the moon icon in the menu bar indicates Focus is on. While active, notifications are silenced.

🪟 Windows 10 / 11

  1. Open Settings (Win + I).
  2. Go to System → Notifications (Windows 11) or System → Notifications & actions (Windows 10).
  3. Make sure Notifications is toggled On at the top.
  4. Scroll down to Google Chrome in the app list and enable it.
  5. Go back to the extension settings and click Send test notification again.
Windows also has a Focus Assist / Do Not Disturb mode. Open the notification panel (bottom-right clock) and check it's not active.

🐧 Linux (GNOME / Ubuntu)

  1. Open Settings → Notifications.
  2. Make sure Do Not Disturb is off.
  3. Scroll to Google Chrome and enable notifications.
  4. Go back to the extension settings and click Send test notification again.
On KDE Plasma, go to System Settings → Notifications → Applications and enable Chrome there.

Still not working?

  1. In Chrome, go to chrome://settings/content/notifications and make sure notifications are not blocked globally.
  2. Try restarting Chrome after changing OS notification settings.
  3. If the issue persists, feel free to open an issue with your OS version and Chrome version.
